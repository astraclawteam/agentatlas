# AgentAtlas

AgentAtlas is the open-core repository for an agent-first enterprise RAG and organizational memory system.

It turns organization spaces, SOP workflows, work brief timelines, dream summaries, and evidence pointers into auditable knowledge workflows for enterprise agents.
